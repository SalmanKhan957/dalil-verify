# Dalil Verify

A citation-first Islamic verifier MVP.

## Phase 1 Scope
- Quran Arabic
- One English Quran translation
- Sahih al-Bukhari
- Sahih Muslim

## Product Boundaries
- Not a fatwa engine
- Not a scholar replacement
- Verifier first, ask engine later

## Immediate Build Order
1. Source registry
2. Quran ingestion
3. Lexical verifier baseline
4. Gold test runner
5. Hybrid retrieval
